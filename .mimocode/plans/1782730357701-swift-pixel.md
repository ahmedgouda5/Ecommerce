# Luxury Ecommerce Redesign Plan

## Overview
Transform the FreshCart ecommerce from a green-themed app into a luxury dark & rose gold experience. Change the auth flow so browsing is public, but cart/checkout/wishlist actions require login. Upgrade react-router-dom to v7.

---

## 1. Update Dependencies

**File:** `package.json`

- Upgrade `react-router-dom` from `^6.23.1` to `^7.6.1` (latest v7)
- Run `npm install` after changes

---

## 2. Tailwind Config - New Color Palette

**File:** `tailwind.config.js`

Replace the primary green with a luxury dark & rose gold palette:

colors: {
